# ✈️ Flight Customer Data Cleaning Automation

## 📌 Project Overview

This project automates the cleaning and preprocessing of flight customer data using Python and Pandas. It identifies missing values, removes duplicate records, standardizes data formats, handles inconsistent values, and prepares the dataset for exploratory data analysis (EDA) and machine learning.

---

## 🎯 Objectives

* Clean raw flight customer data automatically.
* Handle missing and duplicate records.
* Standardize column names and data formats.
* Detect and correct inconsistent values.
* Prepare a clean dataset for further analysis.
* Generate summary reports of the cleaning process.

## 🔄 Data Cleaning Workflow


```bash

```

4. Run all cells.
